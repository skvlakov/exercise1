Pouzil som Eclipse, maven.
-----
Spustenie testu:
mvn test
-----
Install:
mvn install
--------------------------------------
Vznikne target\exercise1-0.0.1-SNAPSHOT.jar:
---
Vstupne nepovinne parametre:
-s cesta k suboru so sumami
-k cesta k suboru s kurzami
-v verbose / logovat priebeh programu
-----
Na vstupe zadat "stop" pre ukoncenie programu.
"teraz" - vypisat vsetky sumy hned.
-----
Spustat takto:
---
java exercise1-0.0.1-SNAPSHOT.jar -v -s subor-sumy -k subor-kurzy
---
Parametre:
   -s nepovinny
   -k nepovinny
   -v verbose - nepovinny
---
